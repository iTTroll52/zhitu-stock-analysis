# 注意力、事件与价格反应

本模块只在获得真实新闻、公告或热点观察时启用。智兔本身不能证明新闻事实或全网热度；缺少真实来源时标记 `not_available`，禁止由模型编造热度、排名或传播路径。

## 来源分层

1. 交易所、政府、监管、公司法定公告；
2. 公司官网和官方投资者关系；
3. 权威媒体线索；
4. 聚合榜单和社交注意力。

前三层用于事实验证，第四层只测量关注度。转载内容先聚类去重，不能把同一原文的多次转载当作独立证据。

## 分开输出四个维度

- `attention`：首次出现、持续时间、排名变化、传播平台和升温/衰减；
- `credibility`：原始来源、官方确认、时间和证据等级；
- `market_confirmation`：板块宽度、成交、涨停/炸板、个股相对强弱和可成交性；
- `crowding`：事前涨幅、极端成交、高开低走、炸板和扩散停止。

热点最多进入题材/市场结构的小部分分值，不能提高财务、订单、产能、收入占比或行业地位分。

## 事件分类

- `positive_new_information`：可信、可量化且未充分定价的新信息；
- `positive_expected`：方向积极但基本符合预期；
- `positive_priced_in`：发布前已经明显交易；
- `positive_realization_risk`：确认后价格负反馈或板块分化；
- `negative_new_information`：新增且可能下修盈利/估值；
- `negative_landing`：坏消息提前交易，确认后损害未继续扩大且价格承接；
- `mixed_or_unknown`：证据冲突或无法量化。

## 必查背离

- 注意力升温，但板块中位数、宽度或成交不确认；
- 价格先涨，新闻后到；
- 文字利好，但发布后高开低走、炸板或相对强度下降；
- 文字利空，但不再创新低、出现承接；
- 单一龙头强，跟随股和流动性中军走弱；
- 热度快速衰减且没有公告、订单、产能或业绩验证。

所有发布后反应必须按5/15/30分钟、收盘、T+1/T+5分开，并严格区分“事件当时可知信息”和“后来结果”。后来结果只能进入复盘，不能回写原判断。
