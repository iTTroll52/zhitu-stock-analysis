# 海外市场、政策与地缘风险层

本层用于判断外部环境如何传导到 A 股，不直接预测涨跌。每项必须记录数据时间、信息发布时间和来源。

## 1. 最小观察集

| 区域/变量 | 观察对象 | 主要传导 |
|---|---|---|
| 美国 | S&P 500、Nasdaq、费城半导体、美国国债收益率、美元指数 | 全球风险偏好、科技估值、融资条件 |
| 日本 | Nikkei 225、TOPIX、日元、日银政策 | 亚洲风险偏好、日元套息、汽车/设备/半导体链 |
| 韩国 | KOSPI、KOSDAQ、半导体/电池龙头、韩元 | 存储、面板、电池、亚洲科技周期 |
| 香港 | 恒生、恒生科技、离岸人民币 | 中国资产情绪和跨境资金映射 |
| 商品 | 原油、黄金、铜及与目标行业有关的原料 | 成本、售价、资源股利润、通胀预期 |
| 地缘/贸易 | 制裁、出口管制、关税、航运与冲突 | 供应链、订单、成本、风险溢价 |

夜盘/海外收盘与 A 股开盘存在时区差。不得把“海外上涨”机械等同为“A 股相关板块上涨”。

## 2. 一手来源优先

优先访问官方原文并链接：

- 中国：国务院、人民银行、证监会、发改委、工信部、商务部、海关总署及交易所；
- 美国：Federal Reserve、US Treasury、White House、USTR、Commerce/BIS、OFAC；
- 日本：Bank of Japan、Ministry of Finance、METI、JPX；
- 韩国：Bank of Korea、Ministry of Economy and Finance、MOTIE、KRX；
- 公司层面：交易所公告、公司官网、法定披露和客户/供应商官方材料。

媒体可用于发现线索，但关键政策、制裁、关税和公司事实必须回到原始文件。社交媒体不得作为唯一证据。

常用官方入口：

- 中国法定披露：[巨潮资讯](https://www.cninfo.com.cn/)、[上交所信息披露](https://one.sse.com.cn/list/xxpl/)、[深交所公司公告](https://www.szse.cn/www/disclosure/notice/company/index.html)
- 美国货币政策：[Federal Reserve Monetary Policy](https://www.federalreserve.gov/monetarypolicy.htm)
- 日本货币政策与日程：[Bank of Japan Monetary Policy](https://www.boj.or.jp/en/mopo/)、[BOJ Release Schedule](https://www.boj.or.jp/en/about/calendar/index.htm)
- 韩国交易所数据：[KRX Data Marketplace](https://data.krx.co.kr/contents/MDC/MAIN/main/index.cmd?locale=en)

这些是入口而非穷尽清单。涉及具体政策时应链接到对应公告原文，而不是只链接首页。

## 3. 传导路径

按链条写结论：

`海外事件 → 价格/政策变量 → A股行业暴露 → 目标公司收入/成本/订单/产能 → 可能的盈利影响`

至少回答：

1. 公司是出口方、进口方、竞争者、客户还是供应商？
2. 受影响业务的收入/成本占比多少？
3. 汇率、关税、替代、库存和合同定价是否削弱或放大影响？
4. 影响是一次性、周期性还是结构性？
5. 公司公告或财报是否确认了该暴露？

如果链条中“公司暴露”没有一手资料支持，只能标记 `行业映射，个股未验证`，不能加基本面分。

## 4. 环境分级

为每个候选给出 `顺风 / 中性 / 逆风 / 高不确定`，并拆为：

- 全球风险偏好；
- 利率与汇率；
- 行业海外映射；
- 政策与地缘事件；
- 数据新鲜度与可信度。

海外与宏观适配最多 10 分。没有明确公司暴露证据时该项最多 4 分；重大地缘事件方向不明时使用 `高不确定`，不要硬判利好或利空。

## 5. 事件冲击处理

- 区分事件发生时间、官方确认时间和市场首次交易时间；
- 检查 A 股是否已经提前交易预期；
- 对限制/制裁同时评估需求损失与国产替代，不能只取一个叙事；
- 对战争或灾害避免从单日价格反应推断长期受益；
- 对海外指数联动使用滚动相关和分阶段比较；相关性不等于因果且会随制度变化。
